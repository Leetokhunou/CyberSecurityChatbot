# Cybersecurity Awareness Chatbot

## Project Overview
This is a C# console-based chatbot designed to educate users about cybersecurity awareness in South Africa. It provides interactive responses about topics such as phishing, passwords, safe browsing, and cyber attacks.

## Features
- Voice greeting on startup
- ASCII art header display
- Personalised user interaction
- Cybersecurity Q&A responses
- Input validation for empty responses
- Colour-coded console UI for better user experience

## How to Run
1. Open the project in Visual Studio
2. Ensure `greeting.wav` is inside the project folder
3. Run the program using Start (F5)

## Technologies Used
- C#
- .NET Console Application
- System.Media for audio playback
